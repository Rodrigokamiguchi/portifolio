Aplicativos

Sistema de Gerenciador de Livros

Aplicativo de controle de biblioteca com funcionalidades de cadastro, empréstimo e devolução de livros.

Desenvolvido com Python, utilizando Tkinter para a interface gráfica e SQLite3 como banco de dados.

Permite gerenciar usuários, visualizar livros cadastrados e acompanhar livros emprestados com datas.

Conversor de Moedas com API

Aplicativo feito com Python e CustomTkinter para converter moedas em tempo real usando uma API externa.

Permite selecionar a moeda de origem e destino, e exibe a cotação atual de forma prática e visual.

O sistema usa arquivos XML para listar as moedas disponíveis e combinações possíveis, consumindo dados da API AwesomeAPI.
